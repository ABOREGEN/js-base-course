# NBSP

## Описание
Написать функцию которая заменяет неразрывные пробелы в тексте управляющим символом `&nbsp`.

## Ввод
- Строка

## Вывод
- Строка

## Тестs

### Тест 1

#### Ввод
```js
[ 'hello world' ]
```

#### Вывод
```
hello&nbsp;world
```

### Тест 2

#### Ввод
```js
[ 'This text contains 4 spaces!!' ]
```

#### Вывод
```
This&nbsp;text&nbsp;contains&nbsp;4&nbsp;spaces!!
```
