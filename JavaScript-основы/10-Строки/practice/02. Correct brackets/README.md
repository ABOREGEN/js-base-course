# Правильные скобки

## Описание
Написать функцию JavaScript, которая проверяет правильность расставления скобок в строке.

## Ввод
- Строка

## Вывод
- Строка
   - `Правильно` или `Неправильно`

## Тесты

### Тест 1

#### Ввод
```js
[ '((a+b)/5-d)' ]
```

#### Вывод
```
Правильно
```

### Тест 2

#### Ввод
```js
[ ')(a+b))' ]
```

#### Вывод
```
Неправильно
```
