# Скажи привет

## Описание
Напишите метод, который запрашивает у пользователя его имя и печатает `Привет, <имя>!`.
Напишите программу для тестирования этого метода.

## Ввод
- На первой строке вы получите имя

## Вывод
- Print `Привет, <имя>!`


## Тест

| Input | Output        |
|:------|:--------------|
| Петя  | Привет, Петя! |
