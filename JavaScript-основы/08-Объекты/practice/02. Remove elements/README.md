# Удалить элементы

## Описание
Напишите функцию, которая удаляет все элементы с заданным значением.
Прикрепите его к типу массива.
Читайте о **прототипе** и о том, как присоединить методы.

```JS
Var arr = [1,2,1,4,1,3,4,1,111,3,2,1,1];
arr.remove (1); // arr = [2,4,3,4,111,3,2];
```

Вам будет задан массив строк. Удалите все вхождения первого элемента во всем массиве.

## Вход
- Вход будет состоять из массива строк

## Вывод
- Выход должен быть массивом с удаленными элементами
   - Каждая строка должна быть отдельной строкой

```

#### Вывод

2
```
