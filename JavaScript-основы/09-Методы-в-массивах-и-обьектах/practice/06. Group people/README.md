# Группа людей

## Описание
- Напишите функцию которая сортирует людей по первой букве имени и выводит отдельным объектом
   - Используйте **Array.reduce()**
   - Используйте только **методы массива**  а не циклы (`for`,` while`)

_Пример:_

```js
result = {
    'a': [{
        firstname: 'Asen',
        /* ... */
    }, {
        firstname: 'Anakonda',
        /* ... */
    }],
    'j': [{
        firstname: 'John',
        /* ... */
    }]
};
```
