# Найти сумму в массиве

## Описание
Напишите программу, которая находит в данном массиве целых последовательность заданной суммы `S` (если она есть).

## Примеры тестов

|        массив           |  S |  результат |
|-------------------------|----|------------|
| 4, 3, 1, **4, 2, 5**, 8 | 11 | 4, 2, 5    |
