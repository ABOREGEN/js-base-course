# Перестановки множества

## Описание
Напишите программу, которая читает число `N` генерирует и печатает все перестановки чисел [` 1 ... N`].

## Примеры тестов

| N |  result   |
|:-:|:---------:|
| 3 | `{1, 2, 3}` <br> `{1, 3, 2}` <br> `{2, 1, 3}` <br> `{2, 3, 1}` <br> `{3, 1, 2}` <br> `{3, 2, 1}` |
