# Матрица чисел

## Описание
Напишите код javascript, который выводит матрицу, как в приведенных ниже примерах, на заданное целое число **N**. Используйте два вложенных цикла.
 
## Ввод
- Ввод будет содержать , целое число **N**.

## Вывод
- См. Примеры.

## Тесты

### Тест 1

#### Ввод
```js
['2']
```

#### Вывод
```
1 2
2 3
```

### Тест 2

#### Ввод
```js
['3']
```

#### Вывод
```
1 2 3
2 3 4
3 4 5
```

### Тест 3

#### Ввод
```js
['4']
```

#### Вывод
```
1 2 3 4
2 3 4 5
3 4 5 6
4 5 6 7
```
